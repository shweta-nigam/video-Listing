# YouTube Assignment 🎬  

## 📌 Overview  
This project is a **YouTube-like UI** built using **HTML, CSS, and JavaScript**. It includes features such as:  
- A sidebar menu with navigation items  
- YouTube UI   
- A video display section  
- A click on the video opens in YuTube 
- responsive
